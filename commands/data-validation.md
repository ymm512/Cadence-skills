# Data Validation - 验证数据格式

在写入Progress或Checkpoint数据到Serena memory之前，验证数据格式、必填字段、值范围等，确保数据质量。

## 使用场景
- 写入Progress数据前验证
- 写入Checkpoint数据前验证
- 数据格式检查

## 调用方式
通过其他Skills自动调用（如checkpoint、status）

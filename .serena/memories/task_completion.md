# 任务完成检查清单 - Cadence-skills

## 文档创建/修改检查清单

由于本项目是纯文档项目，任务完成后需要检查：

### 1. 文档位置检查

- [ ] 文档是否放在 `.claude/` 对应子目录
- [ ] 没有在项目根目录创建文档（README.md 除外）
- [ ] 没有在 `docs/`、`documents/`、`files/` 等目录创建文档

### 2. 文档命名检查

- [ ] 文件名符合 `YYYY-MM-DD_文档类型_文档名称_v版本号.扩展名` 格式
- [ ] 日期使用阿拉伯数字
- [ ] 版本号正确（首次为 v1.0）

### 3. 内容检查

- [ ] 响应使用中文
- [ ] 文档内容清晰完整

### 4. Git 操作（如适用）

```bash
# 检查状态
git status

# 查看变更
git diff
```

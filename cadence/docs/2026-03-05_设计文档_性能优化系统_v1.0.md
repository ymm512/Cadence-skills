## 📊 索引结构设计

### 索引类型

#### 1. 时间索引 (by_time)

**用途**: 按日期快速查询 Checkpoints

**数据结构**:

```yaml
index-{project_id}-checkpoints-by-time:
  "2026-03-05":
    - "checkpoint-user-auth-brainstorm-uuid1"
    - "checkpoint-user-auth-analyze-uuid2"
  "2026-03-04":
    - "checkpoint-user-auth-brainstorm-uuid3"
  "2026-03-03":
    - "checkpoint-api-refactor-design-uuid4"
```

**存储位置**: `index-{project_id}-checkpoints-by-time` (Serena memory)

---

#### 2. 项目索引 (by_project)

**用途**: 按项目快速查询 Checkpoints

**数据结构**:

```yaml
index-checkpoints-by-project:
  "user-auth":
    - "checkpoint-user-auth-brainstorm-uuid1"
    - "checkpoint-user-auth-analyze-uuid2"
    - "checkpoint-user-auth-brainstorm-uuid3"
  "api-refactor":
    - "checkpoint-api-refactor-design-uuid4"
```

**存储位置**: `index-checkpoints-by-project` (Serena memory,全局)

**说明**: 项目索引是全局的,跨所有项目

---

#### 3. 阶段索引 (by_phase)

**用途**: 按阶段快速查询 Checkpoints

**数据结构**:

```yaml
index-{project_id}-checkpoints-by-phase:
  "brainstorm":
    - "checkpoint-user-auth-brainstorm-uuid1"
    - "checkpoint-user-auth-brainstorm-uuid3"
  "analyze":
    - "checkpoint-user-auth-analyze-uuid2"
  "design":
    - "checkpoint-api-refactor-design-uuid4"
```

**存储位置**: `index-{project_id}-checkpoints-by-phase` (Serena memory)

---

### 索引数据模型

```yaml
Index:
  # 索引元数据
  metadata:
    version: "1.0"
    index_type: "by_time" | "by_project" | "by_phase"
    project_id: string | null  # null 表示全局索引
    created_at: timestamp
    updated_at: timestamp

  # 索引数据
  data:
    {key}: string[]  # Checkpoint ID 列表

  # 统计信息
  stats:
    total_keys: number
    total_checkpoints: number
    last_updated: timestamp
```

**示例**:

```yaml
index-user-auth-checkpoints-by-time:
  metadata:
    version: "1.0"
    index_type: "by_time"
    project_id: "user-auth"
    created_at: "2026-03-04T10:00:00Z"
    updated_at: "2026-03-05T14:30:00Z"

  data:
    "2026-03-05":
      - "checkpoint-user-auth-brainstorm-uuid1"
      - "checkpoint-user-auth-analyze-uuid2"
    "2026-03-04":
      - "checkpoint-user-auth-brainstorm-uuid3"

  stats:
    total_keys: 2
    total_checkpoints: 3
    last_updated: "2026-03-05T14:30:00Z"
```

### 索引更新流程

#### 写入 Checkpoint 时更新索引

```python
def update_indices_on_checkpoint_save(project_id, checkpoint_id, checkpoint_data):
    """
    保存 Checkpoint 时更新索引

    Args:
        project_id: 项目ID
        checkpoint_id: Checkpoint ID
        checkpoint_data: Checkpoint 数据

    Returns:
        bool: 是否成功更新
    """
    try:
        # Step 1: 更新时间索引
        update_time_index(project_id, checkpoint_id, checkpoint_data.timestamp)

        # Step 2: 更新项目索引
        update_project_index(project_id, checkpoint_id)

        # Step 3: 更新阶段索引
        update_phase_index(project_id, checkpoint_id, checkpoint_data.phase)

        return True

    except Exception as e:
        log_error(f"更新索引失败: {e}")
        return False
```

#### 更新时间索引

```python
def update_time_index(project_id, checkpoint_id, timestamp):
    """
    更新时间索引

    Args:
        project_id: 项目ID
        checkpoint_id: Checkpoint ID
        timestamp: 时间戳

    Returns:
        bool: 是否成功更新
    """
    # Step 1: 读取时间索引
    index_key = f"index-{project_id}-checkpoints-by-time"
    time_index = read_memory(index_key)

    if not time_index:
        # 创建新索引
        time_index = {
            "metadata": {
                "version": "1.0",
                "index_type": "by_time",
                "project_id": project_id,
                "created_at": get_current_timestamp(),
                "updated_at": get_current_timestamp()
            },
            "data": {},
            "stats": {
                "total_keys": 0,
                "total_checkpoints": 0,
                "last_updated": get_current_timestamp()
            }
        }

    # Step 2: 提取日期
    date = extract_date(timestamp)

    # Step 3: 添加 Checkpoint ID 到对应日期
    if date not in time_index["data"]:
        time_index["data"][date] = []

    time_index["data"][date].append(checkpoint_id)

    # Step 4: 更新统计信息
    time_index["stats"]["total_keys"] = len(time_index["data"])
    time_index["stats"]["total_checkpoints"] = sum(
        len(checkpoints) for checkpoints in time_index["data"].values()
    )
    time_index["stats"]["last_updated"] = get_current_timestamp()

    # Step 5: 更新时间戳
    time_index["metadata"]["updated_at"] = get_current_timestamp()

    # Step 6: 保存索引
    write_memory(index_key, time_index)

    return True
```

## 🔍 查询性能优化

### 查询对比

#### 优化前(全表扫描)

```python
def query_checkpoints_by_date_slow(date):
    """
    查询指定日期的 Checkpoints(慢速)

    Args:
        date: 日期字符串(YYYY-MM-DD)

    Returns:
        list: Checkpoint ID 列表
    """
    # Step 1: 列出所有 Checkpoints
    all_checkpoints = list_memories("checkpoint-*")

    # Step 2: 扫描每个 Checkpoint 检查日期
    result = []
    for checkpoint_key in all_checkpoints:
        checkpoint_data = read_memory(checkpoint_key)
        checkpoint_date = extract_date(checkpoint_data.timestamp)

        if checkpoint_date == date:
            result.append(checkpoint_key)

    # 时间复杂度: O(N)
    # 其中 N = 总 Checkpoint 数量
    return result
```

**性能分析**:
- 时间复杂度: O(N)
- 如果有 1000 个 Checkpoints,需要读取 1000 次
- 预计耗时: 10秒

---

#### 优化后(索引查询)

```python
def query_checkpoints_by_date_fast(project_id, date):
    """
    查询指定日期的 Checkpoints(快速)

    Args:
        project_id: 项目ID
        date: 日期字符串(YYYY-MM-DD)

    Returns:
        list: Checkpoint ID 列表
    """
    # Step 1: 读取时间索引
    index_key = f"index-{project_id}-checkpoints-by-time"
    time_index = read_memory(index_key)

    # Step 2: 查找对应日期
    if date in time_index["data"]:
        return time_index["data"][date]
    else:
        return []

    # 时间复杂度: O(1)
    # 只需要读取 1 次
```

**性能分析**:
- 时间复杂度: O(1)
- 只需要读取 1 次
- 预计耗时: 0.1秒

## 🔄 索引维护

### 索引重建

**场景**: 索引损坏或不一致时重建

```python
def rebuild_indices(project_id):
    """
    重建项目的所有索引

    Args:
        project_id: 项目ID

    Returns:
        dict: 重建统计
    """
    print(f"🔨 重建项目 {project_id} 的索引...\n")

    stats = {
        "time_index": 0,
        "phase_index": 0,
        "project_index": 0,
        "total_checkpoints": 0
    }

    # Step 1: 清除旧索引
    delete_memory(f"index-{project_id}-checkpoints-by-time")
    delete_memory(f"index-{project_id}-checkpoints-by-phase")

    # Step 2: 列出所有 Checkpoints
    all_checkpoints = list_memories(f"checkpoint-{project_id}-*")

    # Step 3: 重建索引
    for checkpoint_id in all_checkpoints:
        checkpoint_data = read_memory(checkpoint_id)

        # 更新时间索引
        update_time_index(project_id, checkpoint_id, checkpoint_data.timestamp)
        stats["time_index"] += 1

        # 更新项目索引
        update_project_index(project_id, checkpoint_id)
        stats["project_index"] += 1

        # 更新阶段索引
        update_phase_index(project_id, checkpoint_id, checkpoint_data.phase)
        stats["phase_index"] += 1

        stats["total_checkpoints"] += 1

    print(f"✅ 索引重建完成:")
    print(f"  - 时间索引: {stats['time_index']} 条")
    print(f"  - 阶段索引: {stats['phase_index']} 条")
    print(f"  - 项目索引: {stats['project_index']} 条")
    print(f"  - 总 Checkpoints: {stats['total_checkpoints']} 条\n")

    return stats
```

### 索引验证

**场景**: 验证索引完整性

```python
def validate_indices(project_id):
    """
    验证索引完整性

    Args:
        project_id: 项目ID

    Returns:
        dict: 验证结果
    """
    print(f"🔍 验证项目 {project_id} 的索引...\n")

    errors = []

    # Step 1: 验证时间索引
    time_index = read_memory(f"index-{project_id}-checkpoints-by-time")
    if time_index:
        for date, checkpoint_ids in time_index["data"].items():
            for checkpoint_id in checkpoint_ids:
                # 检查 Checkpoint 是否存在
                if not memory_exists(checkpoint_id):
                    errors.append(f"时间索引: Checkpoint {checkpoint_id} 不存在")

    # Step 2: 验证阶段索引
    phase_index = read_memory(f"index-{project_id}-checkpoints-by-phase")
    if phase_index:
        for phase, checkpoint_ids in phase_index["data"].items():
            for checkpoint_id in checkpoint_ids:
                # 检查 Checkpoint 是否存在
                if not memory_exists(checkpoint_id):
                    errors.append(f"阶段索引: Checkpoint {checkpoint_id} 不存在")

    # Step 3: 验证项目索引
    project_index = read_memory("index-checkpoints-by-project")
    if project_index and project_id in project_index["data"]:
        for checkpoint_id in project_index["data"][project_id]:
            # 检查 Checkpoint 是否存在
            if not memory_exists(checkpoint_id):
                errors.append(f"项目索引: Checkpoint {checkpoint_id} 不存在")

    # 返回验证结果
    if errors:
        print("❌ 索引验证失败:")
        for error in errors:
            print(f"  - {error}")
        return {
            "valid": False,
            "errors": errors
        }
    else:
        print("✅ 索引验证通过")
        return {
            "valid": True,
            "errors": []
        }
```

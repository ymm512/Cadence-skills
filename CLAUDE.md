## 项目概述

本仓库包含用于编码工作流自动化的 Claude Code 技能（Skills）。技能是模块化增强组件，通过专业知识、工作流或工具集成来扩展 Claude Code 的能力。

## 仓库用途

**Cadence-skills** 用于存储和管理自定义 Claude Code 技能，以改进开发工作流。技能可包括：
- 常见任务的专用命令序列
- 项目特定的工作流自动化
- 领域特定的知识集成
- 自定义工具配置和集成

## 开发方法

### 技能创建原则
创建新技能时，遵循以下原则：
1. **单一职责**：每个技能应只解决一个特定的工作流或功能
2. **明确激活**：技能应有明确定义的触发器（命令、关键词、文件模式）
3. **完善文档**：包含使用示例和预期结果
4. **考虑集成**：考虑技能如何与 MCP 服务器和其他 Claude Code 功能交互

### 文件组织
- 根据领域或用途将技能定义放置在合理的目录中
- 使用描述性命名，清楚表明技能的功能
- 为复杂技能提供 README 文件以解释其架构

# /report - 生成报告

## 使用场景

生成项目进度报告，支持每日报告和周报。

## 功能描述

生成进度报告，包括：
- 今日/本周完成的工作
- 统计数据（工作时长、完成任务、代码提交、测试覆盖率）
- 遇到的问题
- 明日/下周计划

## 命令参数

```bash
/report [--daily | --weekly]
```

- `--daily` - 生成每日报告（默认）
- `--weekly` - 生成周报

## 使用示例

### 每日报告

```bash
/report --daily
```

### 周报

```bash
/report --weekly
```

## 报告生成逻辑

### 每日报告

**数据来源**:
- Session Summary（今日会话）
- Git commit history（今日）
- TodoWrite 任务状态（今日）

**时间范围**: 今日 00:00 - 当前时间

---

### 周报

**数据来源**:
- Session Summaries（本周所有会话）
- Git commit history（本周）
- TodoWrite 任务状态（本周）
- Checkpoints（本周所有）

**时间范围**: 本周一 00:00 - 当前时间
